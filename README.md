# 📌 RBAC Task Management System

A modern **Role-Based Access Control (RBAC) Task Management Application** built using **Node.js, Express.js, MongoDB, and EJS** following the **MVC architecture**.

This application enables organizations to manage tasks and records efficiently with **role-based permissions** for secure and controlled access.

---

- Features

- Authentication & Authorization
  - Secure login system
  - Role-based access control (RBAC)

- User Roles
  - **Admin** → Full access (manage users, records, permissions)
  - **Manager** → Can create & update records
  - **Employee** → Limited or view-only access

- Records Management
  - Create, update, and delete records
  - Modal-based editing UI
  - Clean and responsive layout

- Permission-Based Access
  - Backend route protection
  - Dynamic UI rendering based on permissions

- MVC Architecture
  - Structured and scalable codebase
  - Separation of concerns

- Modern UI
  - Bootstrap 5 + custom styling
  - Responsive dashboard
  - Glassmorphism-inspired design

---

- Tech Stack

- **Frontend:** EJS, Bootstrap 5, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Architecture:** MVC Pattern
